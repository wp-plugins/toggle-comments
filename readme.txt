Toggle Comments allows you to turn WordPress comment functionality on or off.

== Description ==
Inspired by comment 23 on trac ticket [#27111](https://core.trac.wordpress.org/ticket/27111#comment:23) this plugin allows you to:

* Disable support for comments and trackbacks in post types	
* Close comments on the front-end	
* Hide existing comments	
* Remove the admin comments page and comments menu	
* Redirect any user trying to access the admin comments page	
* Remove comments metaboxes from the admin dashboard	
* Remove comments links from the admin bar

== Installation ==

1. Upload `toggle-comments` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Manage functionality under `Settings > Toggle Comments` in the admin
